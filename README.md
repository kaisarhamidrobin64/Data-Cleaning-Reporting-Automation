# Workflow

##  Business Problem

The HR team needed a reliable reporting system to analyze employee data. However, the raw dataset contained missing values, duplicate records, inconsistent formatting, and other data quality issues that could lead to inaccurate reporting and poor business decisions.



##  Collect Dataset

The employee dataset was imported from an Excel file into Power BI, where the data preparation process began.



##  Data Cleaning

Before creating any reports, I cleaned the dataset using Power Query by removing duplicate records, handling missing values, trimming extra spaces, correcting data types, and fixing inconsistent text values.



##  Data Transformation

After cleaning, the data was transformed into a consistent and structured format. Department names, city names, and other categorical values were standardized to improve reporting accuracy.



##  Reporting Automation

Instead of cleaning the data manually every time, I built an automated Power Query workflow. This allows the entire cleaning process to be repeated automatically whenever new data is loaded or refreshed.



##  Power BI Dashboard

An interactive dashboard was created to present key employee insights, including:

- Total Employees
- Average Salary
- Total Departments
- Total Cities
- Employees by Department
- Average Salary by Department
- Employees by City
- Gender Distribution
- Employee Joining Trend
- Interactive Department and City Filters



##  Insights

The dashboard makes it easy to identify workforce distribution across departments and cities, compare average salaries, monitor hiring trends, and understand employee demographics through interactive visualizations.



##  Recommendations

- Maintain consistent data entry standards.
- Validate employee records before importing data.
- Reduce duplicate and incomplete records.
- Refresh the dashboard regularly to keep reports up to date.



##  Business Impact

This project demonstrates how automated data cleaning and interactive reporting can improve data quality, reduce manual work, and support faster, more informed business decisions. It also highlights the importance of building reusable Power Query workflows for efficient reporting.
